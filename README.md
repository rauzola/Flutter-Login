## 🚀 Instalação

1. Clone o repositório
```bash
git clone https://github.com/rauzola/Flutter-Login
```

2. Entre no diretório do projeto
```bash
cd Flutter-Login
```

3. Instale as dependências
```bash
flutter pub get
```

4. Execute o projeto
```bash
flutter run
```
